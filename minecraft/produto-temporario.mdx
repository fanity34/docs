<Warning>
**Lembrando:** O tutorial vale somente para quem quer usar o plugin da CentralCart para remover o beneficio no servidor de forma automática, caso utilize outro plugin para a remoção dos benefícios, não precisa seguir o tutorial abaixo.
</Warning>

## Configuração do produto
Para deixar o produto temporário, você precisa configurar um comando para remover o beneficio depois de um período de dias. Iremos mostrar no exemplo abaixo, como você pode fazer a configuração:

### Exemplo

1. Acesse a configuração do seu produto e determine os dias que o beneficio do jogador será removido, na imagem abaixo, definimos como 30 dias.

<Frame>
  ![Tempo ate que o produto seja removido](/images/minecraft/remove-days.png)
</Frame>

2. Crie um comando que será executado quando o jogador comprar o produto na loja, no nosso caso iremos configurar para adicionar o cargo ouro após a compra. 

<Frame>
  ![Tempo do produto](/images/minecraft/comando-confirmacao.png)
</Frame>

<Info>
Cada servidor pode utilizar um plugin diferente, adicione o comando que o seu servidor utiliza para adicionar os benefícios. 
</Info>

3. Agora para o grupo ouro ser removido depois dos 30 dias, você precisa criar um comando para remover o beneficio ao expirar o produto, igual na imagem abaixo:

<Frame>
  ![Tempo do produto ser removido](/images/minecraft/comandos-minecraft.png)
</Frame>

4. Pronto, o grupo Ouro será removido após os 30 dias.

<Note>
Você pode personalizar tempos diferentes para cada comando ao selecionar a engrenagem e especificar os dias que o comando será executado. 
</Note>

## Conclusão
Comandos com a opção **quando expirar** serão executado conforme a configuração da engrenagem ou por padrão na opção **remover do cliente após**.
